# 📊 KOSPI & KOSDAQ 실시간 지수 시각화 서비스

네이버 증권 API를 활용하여 대한민국 주요 지수(KOSPI, KOSDAQ)의 실시간 변동을 아름답고 직관적으로 시각화하는 Streamlit 애플리케이션입니다.

## 🚀 주요 기능

### 1. 실시간 지수 시각화 (Line Race)
- **KOSPI & KOSDAQ 통합 차트**: 두 지수를 한 차트에서 동시에 비교할 수 있습니다.
- **다중 Y축 (Dual Y-Axis)**: 지수 단위가 다른 두 지수를 효과적으로 비교하기 위해 왼쪽/오른쪽 독립된 Y축을 사용합니다.
- **Race 스타일**: 선의 끝에 현재 지수 값이 따라다니며 시각적으로 역동적인 경험을 제공합니다.

### 2. 정밀한 비교 및 분석
- **09:00 시작점 동기화**: 장 시작 시점(09:00)의 높이를 일치시켜, 지수 절대값과 관계없이 당일의 상대적인 상승/하락 강도를 직관적으로 비교할 수 있습니다.
- **최고/최저점 마커**: 당일의 최고점(빨간색)과 최저점(파란색)을 풍선 아이콘으로 표시하며, 해당 시점의 **시간(HH:MM)**과 지수를 함께 보여줍니다.

### 3. 사용자 최적화 레이아웃
- **고정 X축**: 데이터 유무와 상관없이 09:00부터 15:30까지의 장 전체 시간을 표시합니다.
- **상중 지표 레이아웃**: 현재 지수와 등락 정보를 차트 상단에 가로로 배치하여 가독성을 높였습니다.
- **세련된 디자인**: 현대적인 컬러 팔레트와 가느다란 선(width:1.5)을 적용하여 프리미엄한 디자인을 선사합니다.

## 🛠 기술 스택

- **Frontend/Server**: Streamlit
- **Visualization**: pyecharts (via streamlit-echarts)
- **Data Handling**: Pandas, NumPy
- **API**: Naver Finance Stock API

## 📋 실행 방법

1. **필수 라이브러리 설치**:
   ```bash
   pip install streamlit streamlit-echarts pyecharts requests pandas numpy pytz
   ```

2. **애플리케이션 실행**:
   ```bash
   streamlit run app.py
   ```
